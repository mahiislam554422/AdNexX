HOW TO USE
========================================

1. RUN AS ADMINISTRATOR
   - Right-click on the application
   - Select "Run as administrator"
   - This is required for MAC changes

2. LICENSE ACTIVATION (First Time)
   - Copy your Machine ID from the app
   - Contact developer on WhatsApp
   - Enter license key provided
   - Click "Activate License"

3. CHANGE MAC ADDRESS
   - Select your network adapter
   - Enter new MAC or click "Random"
   - Click "Change Now!"
   - Adapter restarts automatically

4. RESTORE ORIGINAL MAC
   - Click "Restore Original" button
   - Adapter restarts automatically
   - Original MAC is restored
